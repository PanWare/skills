# 小说创作技能

## 核心能力

本技能能够根据用户提供的基本信息，自主设计并撰写完整小说。

## 输入参数（用户提供）

| 参数 | 说明 | 示例 |
|------|------|------|
| 类型 | 小说类型/体裁 | 都市、玄幻、穿越、悬疑、科幻、言情、历史 |
| 主题 | 核心主题 | 复仇、逆袭、成长、爱情、冒险、救赎 |
| 章数 | 预计章节数 | 10-100章 |
| 字数 | 每章/总计字数 | 2000-5000字/章 |
| 风格 | 叙事风格（可选） | 轻松幽默、深沉虐心、热血激昂、文艺清新 |

## 创作流程

### 阶段一：项目设定

当用户提供基本信息时，自动生成以下内容：

1. **小说名称**：富有吸引力的书名
2. **类型定位**：明确的类型和受众
3. **核心卖点**：故事的亮点和特色

### 阶段二：角色设定

为每个主要角色创建详细档案：

| 字段 | 说明 |
|------|------|
| 姓名 | 角色全名 |
| 性别 | 男/女 |
| 年龄 | 角色年龄 |
| 性格 | 性格特点关键词 |
| 背景 | 人物背景故事 |
| 能力 | 特殊技能或身份 |
| 目标 | 角色的核心诉求 |
| 矛盾 | 内在冲突点 |

### 阶段三：世界观设定

描述故事发生的环境：

- 时代背景
- 社会结构
- 关键地点
- 核心规则/力量体系

### 阶段四：小说大纲

按章节拆解故事主线：

```
第一章：[章节名] - 核心事件
第二章：[章节名] - 核心事件
...
```

每章大纲包含：
- 本章核心冲突
- 主要情节点
- 角色发展
- 伏笔埋设

### 阶段五：章节撰写

根据大纲逐步输出完整章节内容：

- 每章包含：开篇、冲突、高潮、结尾
- 合理的情节衔接
- 生动的场景描写
- 符合角色性格的对话

## 输出格式

### 完整项目输出结构

```markdown
# 《小说名称》

## 基本信息
- 类型：都市异能
- 主题：逆袭成长
- 风格：热血激昂
- 预计章节：50章
- 每章字数：3000字

## 角色设定
### 主角
[详细角色卡]

### 重要配角
[配角设定]

## 世界观
[背景设定]

## 大纲
### 第一卷：xxx
- 第一章：[名] - [简介]
- 第二章：[名] - [简介]
...

### 第二卷：xxx
...

## 正文
### 第一章：[章节名]
[完整正文内容]
```

## 写作原则

1. **代入感**：情节真实可信，情绪共鸣强烈
2. **节奏感**：张弛有度，高潮迭起
3. **逻辑性**：伏笔回收合理，动机明确
4. **可读性**：语言流畅，画面感强

## 迭代机制

- 用户可随时要求修改任一部分
- 支持指定风格调整
- 可补充或扩展特定情节
- 接受字数和章节数调整

## 触发方式

用户提供以下任一形式即触发本技能：

- "写一个[类型]小说"
- "帮我创作小说"
- "生成小说大纲"
- "设计角色设定"
- "写第X章"
- "继续写小说"
- "[具体小说想法]"
