# 小说风格指南

## 类型参考

### 都市小说
- 背景：现代都市、职场商场
- 特点：贴近生活、易产生共鸣
- 关键词：逆袭、复仇、商战、职场晋升

### 玄幻小说
- 背景：架空世界、修仙体系
- 特点：想象力丰富、升级体系
- 关键词：修炼、奇遇、秘境、种族战争

### 穿越小说
- 背景：跨时代或跨世界
- 特点：现代知识古代用、知识反差
- 关键词：重生、魂穿、系统

### 悬疑小说
- 背景：现代社会或特定封闭环境
- 特点：推理逻辑、层层反转
- 关键词：破案、密室、连续杀人

### 科幻小说
- 背景：未来世界、外星文明
- 特点：科技感、思考人性
- 关键词：星际、机甲、AI、末日

### 言情小说
- 背景：任意
- 特点：情感细腻、CP感强
- 关键词：甜宠、虐恋、双向奔赴

## 写作技巧

### 开篇写法
1. 制造悬念/冲突
2. 展示主角独特性
3. 建立世界观雏形

### 章节结构
```
开篇（100-200字）→ 铺垫（500字）→ 发展（1000字）→ 高潮（800字）→ 结尾（500字）
```

### 对话技巧
- 每句对话不超过30字
- 用动作/神态打断长对话
- 避免"他说"、"她说"过度使用

### 情绪渲染
- 悲伤场景：环境烘托 + 细节描写
- 热血场景：短句 + 排比 + 紧迫感
- 甜蜜场景：内心独白 + 小动作

## 大纲格式

```
# 《书名》

## 第一卷：XXX（10章）
### 第1-3章：开局
### 第4-6章：发展
### 第7-10章：小高潮

## 第二卷：XXX（10章）
...
```

## 每章大纲模板

```
### 第X章：[章节名]
- **核心事件**：
- **主角行动**：
- **配角参与**：
- **伏笔/悬念**：
- **本章亮点**：
```
