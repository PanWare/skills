## 一、核心效率指标

### 1.1 OEE（设备综合效率）
**公式**：`OEE = 可用率 × 性能率 × 质量率`

**详细计算**：
```
计划生产时间 = 班次时间 - 计划停机时间
实际运行时间 = 计划生产时间 - 非计划停机时间
可用率 = 实际运行时间 / 计划生产时间

理想产量 = 实际运行时间 / 理想节拍时间
性能率 = 实际产量 / 理想产量

质量率 = 合格品数量 / 实际产量
OEE = 可用率 × 性能率 × 质量率 × 100%
```

**六大损失分析**：
| 损失类型 | 影响因子 | 原因分析 | 对策 |
|----------|----------|----------|------|
| 设备故障 | 可用率 | 突发停机、零件磨损 | TPM预防维护 |
| 换型调整 | 可用率 | 批量生产、产品切换 | SMED快速换型 |
| 空转/微小停顿 | 性能率 | 物料堵塞、传感器误报 | 自主保全、标准化 |
| 速度降低 | 性能率 | 设备老化、工艺参数 | 设备升级、参数优化 |
| 工序缺陷 | 质量率 | 工艺不稳定、操作失误 | 防错装置、标准化作业 |
| 启动损失 | 质量率 | 换型后调试、升温 | 首件检验、参数预设 |

### 1.2 节拍时间（Takt Time）
**公式**：`节拍时间 = 可用生产时间 / 客户需求量`

**应用场景**：
- 确定生产节奏和人员配置
- 设计工位数量：`工位数 = 总作业时间 / 节拍时间`
- 识别瓶颈工序

### 1.3 线平衡率（Line Balance Rate）
**公式**：
```
线平衡率 = (各工序时间之和 / (瓶颈工序时间 × 工序数)) × 100%
平衡损失率 = 100% - 线平衡率
```

### 1.4 在制品库存（WIP）
**利特尔法则**：`WIP = 产出率 × 流程时间`

## 二、价值流图（VSM）

### 2.1 关键指标
```
C/T (Cycle Time) = 单件周期时间
C/O (Changeover Time) = 换型时间
Uptime = 设备可用率
增值时间 = 所有工序C/T之和
增值比 = 增值时间 / 总交付周期
```

### 2.2 交付周期计算
```
总交付周期 = 增值时间 + 库存时间 + 等待时间 + 搬运时间
```

### 2.3 VSM绘制步骤
1. 选择产品族
2. 绘制当前状态图
3. 分析问题点
4. 设计未来状态图
5. 制定实施计划

## 三、快速换型（SMED）

### 3.1 换型时间分析
- **内部作业**：必须停机才能进行的作业
- **外部作业**：可以在设备运行时进行的作业

### 3.2 SMED改善五步法
1. **测量**：录像记录换型全过程
2. **分析**：区分内部/外部作业
3. **转化**：将内部作业转化为外部
4. **优化**：简化所有作业
5. **标准化**：制定标准作业书

### 3.3 换型效率指标
```
换型效率改善率 = (改善前时间 - 改善后时间) / 改善前时间 × 100%
```

## 五、看板系统设计

### 5.1 看板数量计算
```
K = (D × L × (1 + α)) / C

K = 看板数量
D = 单位时间需求量
L = 交付周期
α = 安全系数（通常0.1-0.5）
C = 容器容量
```

### 5.2 看板六大规则
1. 后工序只在需要时取货
2. 取货量不超过看板数量
3. 前工序只生产被取走的数量
4. 不允许不合格品流向后工序
5. 看板数量逐步减少
6. 看板适应小幅度需求波动

## 七、TOC约束理论

### 7.1 瓶颈识别
- 资源利用率分析：利用率最高的资源为瓶颈
- 排队分析：WIP堆积最多的工序为瓶颈

### 7.2 DBR生产计划
```
Drum（鼓）：瓶颈工序的生产节奏
Buffer（缓冲）：瓶颈前的安全库存
Rope（绳）：控制投料的信息传递
```

### 7.3 TOC五大聚焦步骤
1. 识别系统瓶颈
2. 挖尽瓶颈潜能
3. 迁就瓶颈决策
4. 打破瓶颈约束
5. 返回步骤1，持续改善

### 7.4 产出会计
```
T (Throughput) = 销售收入 - 完全变动成本
净利润 NP = T - OE
投资回报 ROI = (T - OE) / I
```

## 八、IE工业工程分析

### 8.1 动素分析
**17种基本动素**分为三类：
- 有效动素：伸手、移动、抓取、对准、装配、使用、拆卸、放手、检查
- 辅助动素：寻找、选择、计划、预对
- 无效动素：持住、休息、延迟、故延

### 8.2 标准时间计算
```
标准时间 = 正常时间 × (1 + 宽放率)
正常时间 = 观测时间 × 评比系数
```

### 8.3 方法研究
1. 程序分析：工艺流程图、流程程序图
2. 作业分析：人机作业图、联合作业图
3. 动作分析：动素分析、双手作业分析

### 8.4 人机联合作业分析
```
人利用率 = 人工作业时间 / 周期时间
机利用率 = 机器作业时间 / 周期时间
```

## 十、人员配置与产能规划

### 10.1 人员需求计算
```
人员需求 = 总标准工时 / (有效工作时间 × 效率系数)
```

### 10.2 产能计算
```
日产能 = 有效工作时间 / 单件标准时间
实际产能 = 理论产能 × OEE × 综合效率
```

### 10.3 负荷率分析
```
负荷率 = 实际产量 / 产能 × 100%
```

### 10.4 多能工配置
```
多能工率 = 能操作多工序人数 / 总人数 × 100%
```

## 十一、生产成本分析

### 11.1 成本构成
```
生产成本 = 直接材料 + 直接人工 + 制造费用
```

### 11.2 盈亏平衡分析
```
盈亏平衡点 = 固定成本 / (单价 - 单位变动成本)
安全边际率 = 安全边际 / 实际销量 × 100%
```

### 11.3 成本改善方向
- 材料成本：减少浪费、优化采购、替代材料
- 人工成本：提高效率、自动化、多能工
- 制造费用：减少库存、优化布局、节能降耗

## 十二、质量管理工具

### 12.1 QC七大手法
1. **检查表**：数据收集
2. **柏拉图**：识别主要问题（80/20原则）
3. **特性要因图**：分析原因（鱼骨图）
4. **散布图**：分析相关性
5. **直方图**：分析分布
6. **管制图**：过程监控
7. **层别法**：数据分类

### 12.2 FMEA失效模式分析
```
RPN = 严重度(S) × 发生度(O) × 难检度(D)
优先改善：RPN > 100 或 严重度 > 8
```

### 12.3 SPC统计过程控制
```
控制限计算：
UCL = μ + 3σ
LCL = μ - 3σ

判异规则：
- 1点超出控制限
- 连续7点在中心线一侧
- 连续7点递增或递减
```

### 12.4 MSA测量系统分析
```
GR&R % = (测量系统变异 / 过程变异) × 100%
评价标准：< 10% 优秀，< 30% 可接受，> 30% 需改善
```

### 14.1 生产计划体系

```
MPS = 预测需求 + 订单需求 - 现有库存 - 在途库存
```

```
净需求 = 毛需求 - 现有库存 - 在途库存 + 安全库存
```

## 二十一、操作步骤

1. **需求确认**：明确分析领域和目标
2. **数据收集**：引导用户提供必要数据
3. **计算分析**：精确计算各项指标
4. **对标分析**：对照行业基准识别差距
5. **根本原因分析**：识别问题根源
6. **改善建议**：提供针对性改善方案
7. **实施计划**：制定实施路线图
8. **跟踪验证**：设定目标和跟踪指标

---

## 二十二、注意事项
- 计算前务必确认数据单位和时间基准一致
- 改善建议应结合企业实际情况，分阶段实施
- 所有改善需建立标准化以维持成果
- 重视人的因素，培养精益文化
- **进行计算前必须先读取 [references/formulas.md](references/formulas.md) 确认公式**
- **对标分析时必须先读取 [references/benchmarks.md](references/benchmarks.md)**
- **生成改善建议时必须参考 [references/improvement-cases.md](references/improvement-cases.md)**
- **选择改善工具时必须参考 [references/lean-tools.md](references/lean-tools.md)**
- **制定实施计划时必须参考 [references/implementation-guide.md](references/implementation-guide.md)**
