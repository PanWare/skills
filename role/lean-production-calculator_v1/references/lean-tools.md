# 精益工具使用指南

## 一、工具选择矩阵

### 1.1 按问题类型选择工具
| 问题类型 | 推荐工具 | 适用场景 |
|----------|----------|----------|
| 效率低下 | OEE分析、线平衡、VSM | 设备效率、产线平衡、流程优化 |
| 质量问题 | QC七大手法、FMEA、SPC | 质量分析、预防控制、过程监控 |
| 库存过高 | 看板、VSM、WIP分析 | 库存控制、拉动系统 |
| 换型时间长 | SMED | 快速换型优化 |
| 设备故障多 | TPM、MTBF分析 | 设备管理、预防维护 |
| 布局不合理 | 从至表、P-Q分析 | 布局优化、物流改善 |
| 产能不足 | TOC、产能分析 | 瓶颈识别、产能规划 |
| 成本过高 | 成本分析、VSM | 成本控制、浪费消除 |

### 1.2 按改善阶段选择工具
| 阶段 | 工具 | 目的 |
|------|------|------|
| 诊断 | VSM、OEE、检查表 | 现状把握 |
| 分析 | 鱼骨图、5Why、柏拉图 | 原因分析 |
| 改善 | SMED、ECRS、线平衡 | 实施改善 |
| 控制 | SPC、标准化、看板 | 维持成果 |

## 二、核心工具详解

### 2.1 OEE分析工具
**使用时机**：设备效率问题、产能瓶颈分析

**使用步骤**：
1. 收集数据：计划时间、停机时间、产量、合格品数
2. 计算三率：可用率、性能率、质量率
3. 分析损失：识别六大损失来源
4. 制定对策：针对性改善措施

**注意事项**：
- 数据采集要准确
- 区分计划停机和非计划停机
- 理想节拍时间要合理

### 2.2 价值流图（VSM）
**使用时机**：流程优化、交付周期缩短

**使用步骤**：
1. 选择产品族
2. 绘制当前状态图
3. 计算关键指标
4. 识别问题点
5. 设计未来状态图
6. 制定实施计划

**关键符号**：
- 工序框：矩形
- 库存：三角形
- 信息流：箭头
- 物流：粗箭头

### 2.3 SMED快速换型
**使用时机**：换型时间长、批量生产

**使用步骤**：
1. 测量：录像记录全过程
2. 分析：区分内部/外部作业
3. 转化：内部转外部
4. 优化：简化所有作业
5. 标准化：制定标准作业书

**改善技术**：
- 并行作业
- 快速夹具
- 预设参数
- 可视化标记

### 2.4 看板系统
**使用时机**：库存控制、拉动生产

**使用步骤**：
1. 确定需求量和交付周期
2. 计算看板数量
3. 设计看板形式
4. 建立运行规则
5. 培训实施
6. 持续优化

**看板类型选择**：
- 生产看板：指令生产
- 取货看板：指令搬运
- 信号看板：批量生产

### 2.5 FMEA失效模式分析
**使用时机**：新产品导入、过程改进、风险预防

**使用步骤**：
1. 组建团队
2. 列出失效模式
3. 评估严重度(S)
4. 评估发生度(O)
5. 评估难检度(D)
6. 计算RPN
7. 制定改善措施
8. 重新评估

**评分标准**：
- 严重度：1-10分，10分最严重
- 发生度：1-10分，10分最频繁
- 难检度：1-10分，10分最难检测

### 2.6 SPC统计过程控制
**使用时机**：过程监控、质量稳定

**使用步骤**：
1. 确定控制特性
2. 选择控制图类型
3. 收集数据
4. 计算控制限
5. 绘制控制图
6. 分析判异
7. 采取措施

**控制图选择**：
- 计量值：X-bar R图、X-bar S图
- 计数值：P图、NP图、C图、U图

## 三、工具组合应用

### 3.1 效率改善组合
```
VSM（现状分析）→ OEE（设备效率）→ 线平衡（工序平衡）→ SMED（换型优化）
```

### 3.2 质量改善组合
```
柏拉图（问题识别）→ 鱼骨图（原因分析）→ FMEA（风险预防）→ SPC（过程控制）
```

### 3.3 库存改善组合
```
VSM（流程分析）→ WIP分析（库存诊断）→ 看板（拉动系统）→ 单件流（持续优化）
```

### 3.4 设备改善组合
```
OEE（效率分析）→ MTBF分析（故障分析）→ TPM（设备管理）→ 自主保全（日常维护）
```

- 领导重视与支持
- 全员参与
- 数据驱动
- 持续改善
- 标准化维持

### 4.2 常见问题
| 问题 | 原因 | 对策 |
|------|------|------|
| 数据不准确 | 采集方法不当 | 建立数据采集标准 |
| 工具选择不当 | 不了解工具特点 | 加强培训学习 |
| 实施不彻底 | 缺乏跟进机制 | 建立PDCA循环 |
| 效果不持久 | 标准化不足 | 建立标准化体系 |
