# 精益生产公式速查表

## 一、效率指标公式

### 1. OEE（设备综合效率）
```
OEE = 可用率 × 性能率 × 质量率 × 100%

可用率 = 实际运行时间 / 计划生产时间
性能率 = (实际产量 × 理想节拍时间) / 实际运行时间
质量率 = 合格品数量 / 实际产量

计划生产时间 = 班次时间 - 计划停机时间
实际运行时间 = 计划生产时间 - 非计划停机时间
```

### 2. 节拍时间
```
节拍时间 = 可用生产时间 / 客户需求量

工位数 = 总作业时间 / 节拍时间
人员需求 = 总作业时间 / (节拍时间 × 效率系数)
```

### 3. 线平衡率
```
线平衡率 = (各工序时间之和 / (瓶颈工序时间 × 工序数)) × 100%
平衡损失率 = 100% - 线平衡率
瓶颈损失 = 瓶颈时间 × 工序数 - 各工序时间之和
```

### 4. 在制品库存（WIP）
```
WIP = 产出率 × 流程时间（利特尔法则）
理论WIP = 节拍时间 × 工序数
WIP周转天数 = WIP数量 / 日产量
```

## 二、设备管理公式

### 1. MTBF/MTTR
```
MTBF = 总运行时间 / 故障次数
MTTR = 总修复时间 / 故障次数
设备可用率 = MTBF / (MTBF + MTTR)
```

### 2. 设备效率
```
设备利用率 = 实际运行时间 / 日历时间
设备负荷率 = 实际产量 / 额定产能
```

---

## 三、换型时间公式

### 1. SMED效率
```
换型效率改善率 = (改善前时间 - 改善后时间) / 改善前时间 × 100%
换型损失 = 换型时间 × 换型次数
有效生产时间 = 总时间 - 换型损失
```

---

## 四、看板计算公式

### 1. 生产看板数量
```
K = (D × L × (1 + α)) / C

K = 看板数量
D = 单位时间需求量
L = 交付周期
α = 安全系数（0.1-0.5）
C = 容器容量
```

### 2. 取货看板数量
```
K = (D × (W + S) × (1 + α)) / C

W = 等待时间
S = 搬运时间
```

## 五、六西格玛公式

### 1. 过程能力
```
Cp = (USL - LSL) / (6σ)
Cpk = min[(USL - μ)/(3σ), (μ - LSL)/(3σ)]
Cpu = (USL - μ) / (3σ)
Cpl = (μ - LSL) / (3σ)
```

### 2. DPMO
```
DPMO = (缺陷数 / (机会数 × 单位数)) × 1,000,000
不良率 = DPMO / 1,000,000
```

### 3. 西格玛水平

|--------|------|
| 6σ | 3.4 |
| 5σ | 233 |
| 4σ | 6,210 |
| 3σ | 66,807 |

## 六、TOC约束理论公式

### 1. 产出会计
```
T (Throughput) = 销售收入 - 完全变动成本
I (Investment) = 库存 + 设备投资
OE (Operating Expense) = 运营费用

净利润 NP = T - OE
投资回报 ROI = (T - OE) / I
生产力 = T / OE
```

### 2. 缓冲管理
```
缓冲大小 = 瓶颈周期时间 × 安全系数
缓冲穿透率 = (计划时间 - 实际时间) / 缓冲时间 × 100%
```

## 七、IE工业工程公式

### 1. 标准时间
```
标准时间 = 正常时间 × (1 + 宽放率)
正常时间 = 观测时间 × 评比系数
```

### 2. 宽放率
```
总宽放率 = 个人宽放 + 疲劳宽放 + 延迟宽放
个人宽放：3-5%
疲劳宽放：2-10%
延迟宽放：2-5%
```

### 3. 人机效率
```
人利用率 = 人工作业时间 / 周期时间 × 100%
机利用率 = 机器作业时间 / 周期时间 × 100%
综合效率 = 人利用率 × 机利用率
```

---

## 八、布局与物流公式

### 1. 搬运效率
```
总搬运量 = Σ(物流量 × 距离)
平均活性指数 = Σ(活性指数 × 次数) / 总次数
搬运效率 = 理论搬运时间 / 实际搬运时间
```

### 2. 空间利用
```
空间利用率 = 实际使用面积 / 总面积 × 100%
存储密度 = 存储数量 / 存储面积
```

---

## 九、人员与产能公式

### 1. 人员需求
```
人员需求 = 总标准工时 / (有效工作时间 × 效率系数)
有效工作时间 = 制度时间 × 出勤率 × 有效作业率
```

### 2. 产能计算
```
理论产能 = 有效工作时间 / 单件标准时间
实际产能 = 理论产能 × OEE
产能利用率 = 实际产量 / 产能 × 100%
```

### 3. 负荷分析
```
负荷率 = 实际产量 / 产能 × 100%
超负荷率 = (实际需求 - 产能) / 产能 × 100%
```

## 十、成本分析公式

### 1. 成本构成
```
生产成本 = 直接材料 + 直接人工 + 制造费用
单位成本 = 总成本 / 产量
边际成本 = 总成本变化量 / 产量变化量
```

### 2. 盈亏平衡
```
盈亏平衡点 = 固定成本 / (单价 - 单位变动成本)
安全边际 = 实际销量 - 盈亏平衡销量
安全边际率 = 安全边际 / 实际销量 × 100%
```

### 3. 成本改善
```
成本降低率 = (改善前成本 - 改善后成本) / 改善前成本 × 100%
投资回收期 = 投资额 / 年节约额
```

## 十一、质量管理公式

### 1. FMEA
```
RPN = 严重度(S) × 发生度(O) × 难检度(D)
优先改善：RPN > 100 或 S > 8
```

### 2. SPC控制限
```
X-bar图：
UCL = X̄ + A₂R̄
LCL = X̄ - A₂R̄

R图：
UCL = D₄R̄
LCL = D₃R̄
```

### 3. 过程能力
```
合格率 = Φ(USL-μ/σ) - Φ(LSL-μ/σ)
不良率 = 1 - 合格率
```

## 十二、供应链公式

### 1. 安全库存
```
安全库存 = Z × σd × √L

服务水平对照：
90% → Z = 1.28
95% → Z = 1.65
99% → Z = 2.33
```

### 2. 经济订货量
```
EOQ = √(2 × D × S / H)

D = 年需求量
S = 每次订货成本
H = 单位持有成本
```

### 3. 再订货点
```
再订货点 = 平均日需求 × 交付周期 + 安全库存
```

### 4. 库存周转
```
库存周转率 = 销售成本 / 平均库存
库存周转天数 = 365 / 库存周转率
```
