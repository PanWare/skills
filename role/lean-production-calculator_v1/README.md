# 精益生产全能专家系统 (lean-production-calculator)

精益生产全领域咨询、计算、分析与改善方案设计 Skill。

## 功能概览

| 领域 | 核心能力 |
|------|----------|
| 效率指标 | OEE 计算、节拍时间、线平衡率、WIP 分析 |
| 价值流 | VSM 绘制与关键指标计算 |
| 换型优化 | SMED 五步法、换型效率分析 |
| 设备管理 | TPM 八大支柱、MTBF/MTTR 分析 |
| 拉动系统 | 看板数量计算、看板规则 |
| 质量管理 | QC 七大手法、FMEA、SPC、MSA |
| 六西格玛 | Cp/Cpk、DPMO、DMAIC 流程 |
| 约束理论 | TOC 瓶颈识别、DBR 排程、产出会计 |
| IE 工程 | 动素分析、标准时间、人机联合作业 |
| 布局物流 | 从至表分析、搬运活性、布局类型选择 |
| 人员产能 | 人员需求计算、产能规划、多能工配置 |
| 成本分析 | 成本构成、盈亏平衡、投资回收 |
| 供应链 | 安全库存、EOQ、库存周转 |
| 生产计划 | MPS、MRP、CRP、排程优化 |
| 安全 EHS | TRIR/LTIR、风险评估、5S 与安全 |
| 自动化 | 自动化等级评估、工业 4.0 技术路径 |
| 根因分析 | 5Why、鱼骨图（6M）、问题解决八步法 |
| 精益文化 | 改善提案、人才培养、变革管理 |

## 文件结构

```
lean-production-calculator_v1/
├── SKILL.md                          # Skill 定义与核心指令
└── references/
    ├── formulas.md                   # 公式速查表（计算前必读）
    ├── benchmarks.md                 # 行业基准数据（对标时必读）
    ├── improvement-cases.md          # 改善案例库（生成建议时参考）
    ├── lean-tools.md                 # 精益工具使用指南（选择工具时参考）
    └── implementation-guide.md       # 改善实施指南（制定计划时参考）
```

## 使用方式

1. 将本文件夹复制到 Agent 项目的 Skill 目录
2. 用户提出精益生产相关问题后，Skill 自动触发
3. Agent 按需读取 `references/` 下的参考文档，确保计算与分析准确

## 参考文档使用规则

| 场景 | 必读文档 |
|------|----------|
| 进行任何计算 | `references/formulas.md` |
| 分析结果需对标 | `references/benchmarks.md` |
| 生成改善建议 | `references/improvement-cases.md` |
| 选择改善工具 | `references/lean-tools.md` |
| 制定实施计划 | `references/implementation-guide.md` |

## 操作流程

1. **需求确认** — 明确分析领域和目标
2. **数据收集** — 引导用户提供必要数据
3. **计算分析** — 精确计算各项指标
4. **对标分析** — 对照行业基准识别差距
5. **根因分析** — 识别问题根源
6. **改善建议** — 提供针对性改善方案
7. **实施计划** — 制定实施路线图
8. **跟踪验证** — 设定目标和跟踪指标
